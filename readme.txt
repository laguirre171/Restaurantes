README:

1 - O presente programa pega a lista dos restaurantes que os funcionários inseriram em um arquivo de texto (Local do arquivo: c:\arquivo.txt);
2 - Após carregar o arquivo, os reataurantes são inseridos em uma lista 
3 - É realizado uma escolha randômica do restaurantes

Testes:
1 - É verificado se o arquivo existe;
2 - É verificado se um restaurante está duplicado;
3 - Não pode existir linha em branco entre os restaurantes


O arquivo EscolheRestaurante.java possui a classe main.
